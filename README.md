## 👦 How to play

You can try IchigoJam on the web console of [IchigoJam Web](https://fukuno.jig.jp/app/IchigoJam/).

### 💡 Example Code

#### Turn on LED

```
LED1
```

#### Turn off LED

```
LED0
```

#### Print a random number

```
PRINT RND(10)
```

```
# output
8
```
